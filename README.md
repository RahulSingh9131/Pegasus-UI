# Pegasus-UI
 A component  UI library made using pure HTML and CSS.
 This library contains some useful components which a user can use while making websites faster and easily.
 
 components list
 * Alert
 * Button
 * Avatar
 * Badge
 * Card
 * Input
 * Image
 * Text-Utilities
 * Lists
 * Navigation
 * Modal
 * Rating
 * Snackbar
 * Simplified Grid
